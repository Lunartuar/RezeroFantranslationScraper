A simple Python script that scrapes chapter text from Witch Cult Translations and saves it to a clean .txt file 

Local Installation
Requirements: Python 3.10+
Clone the repo
Install dependencies
Run it
python wct_scraper.py "CHAPTER_URL"

Example:
python wct_scraper.py "https://witchculttranslation.com/2018/09/18/arc-5-chapter-49-thus-opens-the-crusade-against-greed/"

The original Re:Zero web novel belongs to Tappei Nagatsuki
Thank you to WCT for translating this work.

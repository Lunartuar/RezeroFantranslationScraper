# RezeroFantranslationScraper
Project made in Python to Scrape plain text from https://witchculttranslation.com/ and convert it into a plain text file for any needs.


# Local Installation
Requirements: Python 3.10+ 
Clone the repo 
Install dependencies 
Run it python wct_scraper.py "CHAPTER_URL"

# Example:
python wct_scraper.py "https://witchculttranslation.com/2018/09/18/arc-5-chapter-49-thus-opens-the-crusade-against-greed/"
